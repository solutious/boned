## Boned - 0.2 ##

**Bones it!**

## Running 

$ redis-server config/redis-server.conf
$ export BONED_REDIS=redis://localhost:4111/1
$ boned start

## Installation 

$ sudo gem install boned

You also need to install [redis](code.google.com/p/redis/).
