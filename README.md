## Boned - 0.2 ##

**Bones it!**


## Running
    
    $ redis-server config/redis-server.conf
    $ export BONED_REDIS=redis://localhost:8045/1
    $ boned start
    
    
## Installation

    $ sudo gem install boned

You also need to install [redis](http://code.google.com/p/redis/).
